# 20. Valid Parentheses

## Problem Statement

Given a string s containing just the characters '(', ')', '{', '}', '[' and ']', determine if the input string is valid.

An input string is valid if:

1. Open brackets must be closed by the same type of brackets.
2. Open brackets must be closed in the correct order.
3. Every close bracket has a corresponding open bracket of the same type.

### Example 1:
- Input: s = "()"
- Output: true

### Example 2:
- Input: s = "()[]{}"
- Output: true

### Example 3:
- Input: s = "(]"
- Output: false

### Example 4:
- Input: s = "([])"
- Output: true

### Example 5:
- Input: s = "([)]"
- Output: false

### Constraints:
- 1 <= s.length <= 104
- s consists of parentheses only '()[]{}'.

## Solution

### Brute Force (Without using any built-in function)
Intuition: To remove pairs, traverse the whole string at least once (that is why we use do while statement instead of a while statement) for pairs, if we find a matching pair, we remove it (actually replace them by shifting the trailing parentheses towards left). Reduce the length, if there is a change (taken care by toggling of changed variable) then parse the remaining string again. Finally, check if the length of the string is zero or not.

Java
```java
class Solution {
    public boolean isValid(String s) {
        char[] chars = s.toCharArray();
        int n = chars.length;
        boolean changed;

        do {
            changed = false;
            for (int i = 0; i < n - 1; i++) {
                if (isPair(chars[i], chars[i + 1])) {
                    for (int j = i; j < n - 2; j++) {
                        chars[j] = chars[j + 2];
                    }
                    n -= 2;      
                    changed = true;
                    i--;         
                }
            }
        } while (changed);

        return n == 0; 
    }

    private boolean isPair(char open, char close) {
        return (open == '(' && close == ')') ||
               (open == '[' && close == ']') ||
               (open == '{' && close == '}');
    }
}
```

Kotlin

```Kotlin
class Solution {
    fun isValid(s: String): Boolean {
        val charArray = s.toCharArray()
        var n = s.length
        var changed: Boolean

        do {
            changed = false
            var i = 0
            while (i < n - 1) {
                if (isPair(charArray[i], charArray[i + 1])
                 {
                    for (j in i..n-3) {
                        charArray[j] = charArray[j + 2]
                    }
                    n = n - 2
                    changed = true
                    i--
                }
                i++
            }
        } while (changed)

        return n == 0
    }

private fun isPair(open: Char, close: Char): Boolean {
        return (open == '(' && close == ')') ||
               (open == '[' && close == ']') ||
               (open == '{' && close == '}');
    }
}
```

### Brute Force with built-in function
Intuition: Same as above just that we are using a built-in function to replace pairs with empty string.

Java
```java
class Solution {
    public boolean isValid(String s) {
        int length;
        do {
            length = s.length();
            s = s.replace("()", "")
                 .replace("[]", "")
                 .replace("{}", "");
        } while (s.length() < length); 

        return s.isEmpty(); 
    }
}
```

Kotlin
```Kotlin
class Solution {
    fun isValid(s: String): Boolean {
       var n: Int
       var str = s

       do {
            n = str.length
            str = str.replace("()", "")
                .replace("{}", "")
                .replace("[]", "")

       } while (str.length < n)

        return str.length == 0
    }
}
```

### Time and Space Complexity
#### Time Complexity: O(n^2)
#### Space Complexity: O(1)

### Optimized Solution
Intuition: Remove pairs from the top of the stack, if at the end stack is empty then it is balanced, else it is not.

Java
```java
class Solution {
    public boolean isValid(String s) {
        Deque<Character> stack = new LinkedList<>();

        for (char c : s.toCharArray()) {
            if (c == '(' || c == '{' || c == '[') {
                stack.push(c);
            } else {
                if (stack.isEmpty()) return false;
                else {
                    if ((c == ')' && stack.peek() == '(') ||
                        (c == '}' && stack.peek() == '{') ||
                        (c == ']' && stack.peek() == '['))
                        {
                            stack.pop();
                        }
                    else return false;
                }
            }
        }

        return stack.isEmpty();
    }
}
```

### Time and Space Complexity
#### Time Complexity: O(n)
#### Space Complexity: O(n), max stack size if all are open brackets.
